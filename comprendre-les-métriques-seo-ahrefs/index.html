Comprendre les métriques SEO ahrefs

Dans cet article, nous allons voir comment lire les métriques tirées de ahreafs pour comprendre le positionnement de certains domaines.



Bloc du mot-clé

En haut, pour la requête what is seo


KD 96 Keyword Difficulty

Difficulté estimée pour se positionner en top 10 sur ce mot-clé.
Échelle 0-100.
96 = extrêmement difficile. Il faudra probablement beaucoup d’autorité et de backlinks.


Volume 31K

Volume mensuel estimé dans le pays sélectionné, ici probablement les États-Unis car il y a le drapeau US.
Donc environ 31 000 recherches/mois aux US.


GV 98K Global Volume

Volume mensuel mondial estimé
Ici, environ 98 000 recherches/mois dans tous les pays.


TP 59K Traffic Potential

Potentiel de trafic total estimé si tu te positionnes bien sur ce sujet.
Ce n’est pas seulement le mot-clé exact. Ahrefs additionne aussi les variantes proches et mots-clés liés que la page peut capter.


Clicks 16.86K

Nombre estimé de clics organiques mensuels générés par cette SERP.
Important: un mot-clé peut avoir 31K recherches mais seulement 16.86K clics, car certains utilisateurs trouvent la réponse directement dans Google.


CPC 0.70$

Coût par clic moyen estimé en Google Ads.
Plus le CPC est élevé, plus l’intention commerciale est souvent forte. Ici, 0.70$ est plutôt faible/modéré.


Bloc WEBSITE

Ces métriques concernent le domaine entier, pas seulement la page affichée.

Pour developers.google.com:


DR 95 Domain Rating

Autorité globale du domaine selon Ahrefs, basée surtout sur son profil de backlinks.
Échelle 0-100.
95 = domaine ultra puissant.


AR 45 Ahrefs Rank

Classement mondial Ahrefs du domaine selon la puissance de son profil de liens.
Plus le chiffre est bas, plus le site est fort.
45 = énorme autorité mondiale.


RP 115.1M Referring Pages

Nombre de pages externes qui font un lien vers ce domaine.
Ici: environ 115,1 millions de pages.


RD 1.2M Referring Domains

Nombre de domaines différents qui font au moins un lien vers ce site.
Ici: 1,2 million de domaines référents. C’est énorme.


KW 1.4M Key words

Nombre de mots-clés organiques sur lesquels le domaine est positionné dans la base Ahrefs.
Ici: 1,4 million de mots-clés.


ST 2.7M Search Traffic

Trafic organique mensuel estimé du domaine.
Ici: environ 2,7 millions de visites SEO/mois.


2.2M$ sous ST Traffic Value

Valeur estimée de ce trafic si tu devais l’acheter via Google Ads.
Ici: environ 2,2 millions $/mois.


Bloc PAGE

Ces métriques concernent uniquement l’URL affichée dans le résultat Google, pas tout le domaine.

Pour la page Google Developers:


UR 49 URL Rating

Autorité de cette page précise selon Ahrefs.
Échelle 0-100.
49 = page forte.


RP 163K

Nombre de pages externes qui pointent vers cette URL précise.


RD 13.9K

Nombre de domaines différents qui font un lien vers cette page précise.
C’est très élevé.


KW 38.6K

Nombre de mots-clés sur lesquels cette URL précise se positionne.


ST 241.2K

Trafic organique mensuel estimé de cette URL précise.


437.7K$

Valeur estimée du trafic organique de cette page si acheté en Ads.
